CRUCIBLE

AMAZON GAME STUDIOS

UX DESIGNER I & II

WHAT

Crucible is a MOBA Hero Shooter. As the only UX designer on the team for the first year, I wore many hats: bringing a UX culture to the studio by understanding player behavior through user research, designing the underlying UI/HUD structure, managing UI art outsourcing, clarifying game systems and 3D art assets, and improving workflows between UX design and the rest of the studio.

Find more info here: playcrucible.com

HIGHLIGHTS

  • Designed underlying systems for 70 features/components resulting in 400+ individual wireframes. These included HUD elements & features like ranked mode, battle pass, custom games, home lobby, post-match, etc.
  • Created weekly usability sessions at the studio for 9 continuous weeks. This generated 105 issues on our tutorial alone, which helped us prioritize fixes, and improve the FTUE (first-time user experience) funnel.
  • Created 60+ UI art specs for our UI art outsourcing team. Gave detailed feedback on over 500+ individual screens.
  • Designed and ran targeted user research for specific in-game art assets and icons to improve readability and understandability.
  • Designed our colorblind accessibility feature using data from interviews with disabled gamers.
